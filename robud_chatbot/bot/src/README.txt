Source Folder
==============

This folder is the root folder ( and also module name ) of all the various extensible options you want to build. See the
README.txt in each folder for a description of how to build the neccassary Python object

    o dynamic       - Contains all dynamic sets, maps and variables
    o extensions    - Contains all of the extensions
    o oob           - All Out of Band classes
    o processors    - Any additional pre and post processors you want to add
    o security      - Authorisation and Authentication systems you want to add
    o services      - Any SRAIX based services you develop
    o spelling      - Spelling modules to replace the in built on
